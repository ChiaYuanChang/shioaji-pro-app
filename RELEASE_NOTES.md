## v0.1.26 — TradingView 級圖表指標系統

### 指標系統全面重做
- **21 種指標**：主圖 MA / EMA / WMA / BOLL / VWAP / SAR / SuperTrend / Donchian / Keltner；副圖 MACD / RSI / KD(9,3,3 台股慣用) / StochRSI / CCI / ATR / OBV / MFI / W%R / DMI-ADX / ROC / BIAS 乖離率。副圖指標各佔獨立窗格，含參考水平線與 MACD 紅綠柱。
- **TradingView 式指標選擇器**：大型搜尋對話框（中英文、縮寫、別名都能搜，打「乖離」找得到 BIAS）、分類側欄、我的最愛星號收藏、每列附說明；點選先開設定預覽、確定才加入。
- **完整設定視窗**：輸入 / 樣式 / 時框顯示三分頁 — 80 格色盤 + 自訂 hex + 透明度 + 線寬、每條輸出可切線型（線/階梯線/面積/柱狀/圓點）與單獨顯示開關、小數位數、價格軸標籤；「存為我的預設」讓之後新增同型指標直接套用你的參數配色。
- **圖上 legend**：主圖疊加類在主圖左上、副圖指標在各自窗格左上，顯示即時數值（跟著十字線走）；hover 出現隱藏/設定/移除/更多控制，⋯ 選單支援複製指標、調整視覺順序、加入最愛。
- 同型指標可多開（三條不同週期 MA 各自設定）；設定跨重啟保存；拖好的上下窗格比例在改參數/移除指標後精確保留。

### K 線
- **歷史無限左滑**：往左拖到邊緣自動載入更舊的 K 棒（TradingView 式動態分頁，最多回溯三年）。
- **試撮修正**：試撮（simtrade）揭示的漲跌停天地價不再畫進即時 K 棒撐爆座標。(#5)

### 伺服器版本握手
- App 內建 sidecar 版本為唯一基準：桌面版只啟動/銜接與打包版本一致的 server，版本不符的外部 server 不再誤接（會另起內建伺服器，不動你自己跑的）；連到版本不符的 server 時標題列顯示琥珀色警示。
