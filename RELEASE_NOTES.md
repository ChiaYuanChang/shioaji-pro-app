## v0.1.14 — 鋪單、AI 助理、閃電全開與社群回饋（#2）

### 重要修正：更新後卡在「連線中」
- 啟動不再沿用壞掉的舊伺服器：不健康（如正式環境缺憑證）或模式不符（設定模擬、卻在跑正式）會自動停掉重啟
- 啟動／重啟成功後等健康檢查通過自動重新載入畫面（#2 回報的重啟後行情不動）
- 管理面板紅字明示模式不符與不健康的原因和解法

### 新交易功能
- **鋪單面板**：一鍵在多個價位鋪限價單（檔距／檔數／間隔／每檔量可調）、一鍵全撤；**動態跟隨**模式每 2.5 秒自動撤舊補新，讓鋪單跟著現價走（有節流與上限保護）
- **⚡ 全開**：一鍵把自選前 9 檔的閃電下單外開成獨立視窗、自動平鋪滿螢幕
- **AI 助理面板**：接 Claude 查行情、分析持倉、擬單 — 所有下單提案都要手動按確認才會送出（API Key 存本機）
- **組合單到價監控**（#2）：合成價到目標自動嘗試 IOC，最多 3 次、5 秒冷卻、防重複成交警語

### 新分析面板（#2）
- **類股熱力圖**：依商品分類選類股，色塊深淺＝漲跌幅、依成交額排序、點擊連動
- **選擇權損益圖**：依持倉畫到期損益曲線，可勾選部位、加模擬腳（Call/Put/期貨）、標示履約價與現價（結算試算，不含時間價值，盈虧自負）
- **排行榜複選**：漲幅／量／額門檻交叉篩選，找得到排行榜前段以外還能交易的標的

### 回饋修正（#2）
- 契約價值改用 API 乘數（股票期貨先前被當指數期貨算）、選擇權權利金與各商品期交稅率
- 選擇權代碼可解析：T 字點擊後下單面板／閃電可直接交易
- 持倉股票改以股數計算：**零股不再被無條件捨去**，平倉自動拆整張＋零股單；新增**報酬率%**欄
- 自選清單可用**中文股名搜尋**（正股優先於權證）
- 隱私新增**遮金額**：水位／數量／損益／權益一鍵遮蔽
- 主題同步原生視窗外觀（macOS／Windows 標題列跟著深淺色）
