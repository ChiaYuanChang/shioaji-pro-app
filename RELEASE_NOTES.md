## v0.1.27 — 自訂指標 ＋ 策略回測

### 自訂指標：指標自己寫
- **面板內指標編輯器**：用 JavaScript + `ta.*` 函式庫（20+ 個均線/統計/動能/波幅/交叉函式）寫自己的指標，`plot()` 宣告輸出線、`hline()` 畫參考水平線。
- **沙箱驗證**：儲存前在 Web Worker 跑樣本資料驗證，語法錯誤給明確訊息、無窮迴圈 2 秒自動斬掉，不會拖垮終端。
- **與內建指標同等待遇**：自動偵測輸出線後可調顏色/線型/名稱，設定視窗、樣式、時框顯示、收藏、複製全套沿用；指標選擇器新增「自訂」分類與編輯入口。

### 策略回測（桌面版專屬）
- **策略也是自己寫**：`longEntry()` / `longExit()` / `shortEntry()` / `shortExit()` 四種訊號收集器，同一套 `ta` 函式庫，參數表讓回測前免改碼調參。
- **含成本、無未來函數的引擎**：訊號收盤成立、次根開盤成交；股票手續費（可設折扣）/證交稅（ETF 減半）、期貨固定費＋期交稅、滑價按各商品 tick 表；反向訊號自動平倉翻單。
- **多商品整合**：整個自選清單一次跑（低併發抓資料保護 API），合計損益/整體勝率/合併權益曲線＋可排序的各商品績效表，點列展開單商品明細。
- **進出場畫在 K 線上**：回測面板內嵌完整 K 線，▲多 ▼空 ●平標記（可設定顯示方向/數量/成交價），點交易明細直接縮放到那筆進出場；與主圖共用指標設定、即時同步。
- 完整績效指標：報酬率/勝率/獲利因子/最大回撤/每筆期望值/成本合計/持倉占比；權益曲線含未平倉市值。
